# This is WETA

#### It's short for worst ever testing application

#### Instructions:

All you need to do is put in your message and expected value. then call the function. That's it.

## Want an example?

```javascript
let test = Weta('1 + 2 should equal 3', 3);

const add = (a, b) a + b;

test(add(1,2));

```

So that's it. Enjoy!
